How to Play

Quick rundown before you start

  • Every leaf in a round is the same color — either all green or all yellow — and every leaf shows an arrow.
  • Green round: the arrows are honest — answer with the direction each leaf is pointing.
  • Yellow round: the arrows are decoys — ignore them and answer with the direction the leaf is actually drifting instead.
  • Getting one right immediately reshuffles every leaf you haven't answered yet, so don't scout ahead.
  • Resolve each leaf in turn using the arrow keys, WASD, or the on-screen d-pad — before its timer runs out.
