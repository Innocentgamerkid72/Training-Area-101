# Menu bar
Shows the categories of what do you want to train at.
Catergories - 
- Clicker Accuracy
- Clicking Speed
- Reaction time
- Timing
- Arcade
- Typing accuracy
Each category can be activated by the push of a color coded button. Each one sending the user to a different page. The page after selecting a category is a settings page. The setting page can be changed depending on the category chosen.

# Clicker Accuracy
If the user selects the Clicker Accuracy training category, they will first be sent to the settings page. The settings page will ask you if you want the clicker accuracy test to be an array or random popups or even both. If it is an array, the page will then ask you how many rows and columns would you like and then ask you for the size and shape of the target. There will be a small panel for you to decide the size. The panel will decide the shape of the target. You can delete, add and move the corners of the shape in the panel. It will start off as a square. The entire panel is the maximum size the target can be. You can even add another variant of the target. You can also add a don't shoot target variant. If the user picks the random popups mode, they will automatically be sent to the target settings. If the user selects both, the process is similar to if the user selects the array option but they will also choose where will the random popups go and the array. After the target settings is finished, the user will be sent to a timer settings. Where the user will have to adjust how common will the targets popup and how long will it last. After the user verifies everything is final, the game will begin. Following every single instruction the user has asked. After the time is up, the website will say congratulations to the user and then send you back to the home page.

# Clicking Speed
If the user selects the Clicker Speed training category, the user will be sent to a selection page. The selection page will ask for how long will the test last. The user can set it from 5 seconds all the way to 5 minutes. After confirming the time, the clicker test is set up and a 3 second countdown begins. After that, the clicker test begins and the User must clicker on a black square to gain clicks. After the time is up, the website will calculate the clicks per minute the user did and then shows the result. After that, you can go back to the home page.

# Reaction Time
If the User chose the Reaction Time training category, the user will be sent to a selection page. The selection page will first ask the user for a theme. There are 2 themes. The first theme is where the page will a red screen. But before the user need to decide what key the user need to press to react. After deciding the user will be sent to the red page, and react when it turns green. After that the user will be given a score. If the user were to react too early, the user will be forced to restart. The second theme is the standoff. Where it works like the first theme but instead it is a standoff. The user need to react faster than the AI. The AI will give the user 1-3 seconds to react. The signal to react will be instead a large text in the middle. The user can adjust the size of the text. It will first say "Wait..." and when it wants the user to react, it will say "SHOOT!" After that, the user will be given the score of reaction time and can go back to the home page.

# Typing Accuracy
If the user were to choose the Typing Accuracy training category, the user will be sent to the setting page. Where you have to pick how long will you be typing. Then the user must pick a theme. It can be like school essay about science or english. Text from a book. Or even something random. After that, a 3 second countdown shall begin and the typing shall start. If the user made a mistake, it will be recorded into the score but the user can fix that mistake to reduce loss of points. The user must follow a given prompt the user must type. The typo will be highlighted in red but will be yellow once corrected. Any keys that were typed without mistakes will be green. After that, it will calculate the Words per Minute and share the score to the user.

# Timing
If the user were to select the Timing training category, they will to first choose the speed, how long what keybind to use to time, and the size of the both the timing bar and the timing slot. After confirming the settings, the screen will show a bar showing either horizontally or vertically across the website based on what the user say in the settings. The user must use the keybind they selected when a white square aligns will the specific point in the line or timing slot based on what you chose. After the time is up, the user will get a score based on the misses, the fails, and the landings they have done.
